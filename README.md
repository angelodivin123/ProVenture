# ProVenture
ProVenture is an educational maze adventure game where players choose characters, battle enemies, answer quizzes to collect materials, and defeat a final boss to build a ship and escape an island.


# Installation Instructions
1. Install Python 3.8 or higher from python.org
2. Install required libraries following these steps:
   - Open command prompt or terminal
   - Run these commands one by one:
     " pip install pygame "
     " pip install pillow "
3. Download the game files and ensure all resources are in the resources folder
4. Run the game

# Game Features
- Three unique character classes with special abilities
- 30 educational quizzes across multiple difficulty levels
- Boss battle system with quiz-based combat
- Resource collection and ship building mechanics
- Leaderboard system to track high scores

# Controls
- WASD: Move character
- Left Click: Attack enemies
- E: Interact with objects
- ESC: Pause game

Note: Make sure all game resource files are in the correct directory structure.
